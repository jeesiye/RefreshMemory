`java.io.FileInputStream`,可以说是大多数输入流的中流砥柱.
- 在`java.io`包中,`RandomAccessFile`是有自己独立的实现来读取资源的.  
- 其他大多数的输入流都是基于该文件输入流,进行包装扩展功能的.  
- 有几个比较特殊的功能型输入流,使用上不是一定要以来此文件输入流的实例的.
