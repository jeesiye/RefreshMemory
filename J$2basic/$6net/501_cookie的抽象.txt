概念层的概念划分
-----------------------------------------------------------------------------------------------------------------------------
1. 定义规则;
       I:java.net.CookiePolicy
2. 存储方式;
       I:java.net.CookieStore
3. 处理方式;
       A:java.net.CookieHandler
       FL:java.net.CookieManager
-----------------------------------------------------------------------------------------------------------------------------

继承关系图谱
-----------------------------------------------------------------------------------------------------------------------------
CookiePolicy
CookieStore
CookieHandler
        |   extends
        |-------------> CookieManager
-----------------------------------------------------------------------------------------------------------------------------

注意事项
-----------------------------------------------------------------------------------------------------------------------------
HttpCookie 是被CookieManager使用到的;
它和此处说的三个抽象类没有任何的继承关系的;
-----------------------------------------------------------------------------------------------------------------------------


I:java.net.CookiePolicy
-----------------------------------------------------------------------------------------------------------------------------
(内置常量)(应是大写形式,便于阅读改为小写)
-----------------------------------------------------------------------------------------------------------------------------
accept_all                                                         接受所有的cookie;
accept_none                                                        不接受任何cookie;
accept_original_server                                             只接受来自于服务端的cookie;
-----------------------------------------------------------------------------------------------------------------------------
(对外函数)
-----------------------------------------------------------------------------------------------------------------------------
shouldAccept <p2> : boolean                                        检查是否接受指定的cookie;
-----------------------------------------------------------------------------------------------------------------------------


I:java.net.CookieStore
-----------------------------------------------------------------------------------------------------------------------------
add <p2> : void                                                    添加;
get <p1> : List                                                    获取;
getCookies <p0> : List                                             获取cookie集;
getURIs <p0> : List                                                获取uri集;
remove <p2> : boolean                                              删除;
removeAll <p0> : boolean                                           删除所有;
-----------------------------------------------------------------------------------------------------------------------------


A:java.net.CookieHandler
-----------------------------------------------------------------------------------------------------------------------------
getDefault <p0> : CookieHandler                                    获取默认的cookie处理器;(static)
setDefault <p1> : void                                             设置默认的cookie处理器;(static)
get <p2> : Map                                                     获取cookie映射集;
put <p2> : void                                                    添加cookie键值对;
-----------------------------------------------------------------------------------------------------------------------------


FL:java.net.CookieManager
-----------------------------------------------------------------------------------------------------------------------------
put <p2> : void                                                    添加;
get <p2> : Map                                                     获取;
getCookieStore <p0> : CookieStore                                  获取CookieStore对象;
setCookiePolicy <p1> : void                                        设置CookiePolicy;
-----------------------------------------------------------------------------------------------------------------------------
